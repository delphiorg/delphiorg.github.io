Sometimes being on-site slows you down.  The WiFi is spotty and just finding a change to sit down is a challenge.  The news about Delphi Prism is out.  What is Delphi Prism?  It is Delphi with full .NET 3.5+ support running in Visual Studio, with a number of the high power database solutions you would expect from CodeGear / Embarcadero.

Behind the scenes, this is accomplished with a stack of <a href="http://msdn.microsoft.com/en-us/library/bb685612.aspx"></a>
<ul>
	<li><a href="http://msdn.microsoft.com/en-us/library/bb685612.aspx">Microsoft's Visual Studio Shell</a> (which I talked with <a href="http://blog.marcocantu.com/">Marco Cantu</a> about in <a href="http://delphi.org/2008/10/episode-10-marco-cantu/">Episode 10</a>),</li>
	<li><a href="http://www.remobjects.com/oxygene">RemObject's Oxygene Compiler</a> of the Delphi Language for .NET (Talk with <a href="http://blogs.remobjects.com/mh">marc hoffman</a> about in <a href="http://delphi.org/2008/09/episode-5-oxygene-and-net/">Episode 5</a>)</li>
	<li>CodeGear's dbExpress, DataSnap and Blackfish SQL database solutions (Podcasts coming on these topics!)</li>
</ul>
You put all that together and you have a pretty exciting solution for .NET development.

Check out the <a href="http://prismwiki.codegear.com/wiki/Delphi_Prism_vs._CSharp">impressive list of language features</a> available in Delphi Prism, that are not available in C#.  We talked about a lot of these in <a href="http://delphi.org/2008/09/episode-5-oxygene-and-net/">Episode 5</a>.  Notably:
<ul>
	<li><a title="Parallel Loops" href="http://prismwiki.codegear.com/wiki/Parallel_Loops">Parallel Loops</a></li>
	<li><a title="Inline Property Accessors" href="http://prismwiki.codegear.com/wiki/Inline_Property_Accessors">Inline Property Accessors</a></li>
	<li><a title="Class Contracts" href="http://prismwiki.codegear.com/wiki/Class_Contracts">Class Contracts</a></li>
	<li><a title="Constructors" href="http://prismwiki.codegear.com/wiki/Constructors#Extended_Constructor_Calls">Extended Constructor Calls</a></li>
	<li><a title="Boolean Double Comparison" href="http://prismwiki.codegear.com/wiki/Boolean_Double_Comparison">Boolean Double Comparison</a></li>
</ul>
Not to mention all the cool things you can do with the database tools.

I am sure there is much more to come about Delphi Prism.
