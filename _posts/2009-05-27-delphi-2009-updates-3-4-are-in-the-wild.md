The long anticipated <a href="http://edn.embarcadero.com/article/39390">Delphi 2009 Update 3</a> is now available.  Just run check for updates. As if the long list of <a href="http://edn.embarcadero.com/article/39387">fixes in Update 3</a> wasn't enough, they also released a <a href="http://edn.embarcadero.com/article/39392">Database Pack Update 4</a>.  You can also <a href="http://cc.embarcadero.com/item/26921">download to install</a> on multiple machines.  Looks like there is a Boost update for C++ Builder 2009 users too.

<strong>Update:</strong> <a href="http://blogs.embarcadero.com/nickhodges/">Nick Hodges</a>, Delphi R&amp;D manager has added a <a href="http://blogs.embarcadero.com/nickhodges/2009/05/27/39241">post about the release</a>.
