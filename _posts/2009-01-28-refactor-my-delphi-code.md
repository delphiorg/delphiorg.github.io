I recently discovered the fascinating site <a href="http://refactormycode.com/">Refactor My Code</a> where software developers can collaborate on the evolution of a piece of code.  Well, thanks to the work of <a href="http://macournoyer.wordpress.com/category/refactormycode/">Marc-André Cournoyer</a> it now has a <a href="http://refactormycode.com/codes/recent/delphi">section for Delphi code refactorings</a>.

They are using Ruby's <a href="http://ultraviolet.rubyforge.org/">Ultraviolet</a> syntax highlighter, which has a <a href="http://refactormycode.com/help/code">Pascal syntax</a>.  Based on preliminary tests it appears to work correctly.  Let me know if you encounter any thing that doesn't get highlighted correctly and we can work to correct it.

I'll be posting code snippits and looking to help refactor yours!
